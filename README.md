# DM-PROJECT
DM Project (2025-CS-62 AND 2025-CS-64)
Title: Book Serial ID Generation and Verification System
Description: It creates a serial id for entered book using concepts of number theory. It also uses a check digit to verify a valid id.
Usage Examples: It can be used in a library to keep track of changes of books.
Dependencies: It depends on modular arithmetic and concepts of number theory.
